## рҹҢҗ Deployment

### GitHub Pages (Automatic)
1. Configure GitHub repository
2. Enable GitHub Pages (gh-pages branch)
3. Push to main branch
4. GitHub Actions deploys automatically
5. Site live at: `https://username.github.io/ai-interview-academy/`

### Manual Deployment
```bash
npm run build
npm run deploy
```

## рҹ“қ Configuration Updates

Before deploying, update:

**docusaurus.config.js**:
```javascript

projectName: 'ai-interview-academy'
baseUrl: '/ai-interview-academy/'
deploymentBranch: 'gh-pages'
```

**Repository Settings**:
- Enable GitHub Pages
- Set deployment branch to `gh-pages`
- Ensure workflows have permissions
