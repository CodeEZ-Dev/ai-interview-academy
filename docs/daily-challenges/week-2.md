## Common Questions
**Q: How do you evaluate RAG system quality?**
A: Use retrieval metrics (precision, recall) + generation metrics (BLEU, ROUGE) + human evaluation

**Q: How to scale to millions of documents?**
A: Use vector databases, distributed retrievers, caching, and hierarchical retrieval

**Q: How to handle real-time updates?**
A: Implement incremental updates, maintain separate indices, use change data capture
