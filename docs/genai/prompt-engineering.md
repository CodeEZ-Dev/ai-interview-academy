# Prompt Engineering

## Question

What is prompt engineering and why is it critical for LLM applications?

## Answer

**Prompt engineering** is the practice of crafting and optimizing inputs to LLMs to achieve desired outputs. It's a critical skill because LLM behavior is heavily influenced by how questions are framed.

### Key Techniques

1. **Few-shot Learning** - Provide examples in the prompt
2. **Chain-of-Thought** - Ask model to explain reasoning
3. **Role-playing** - Ask model to assume a persona
4. **Constraint Setting** - Define output format and constraints
5. **Iterative Refinement** - Test and improve prompts

### Best Practices

✅ Be specific and clear  
✅ Provide context and examples  
✅ Use structured formats  
✅ Specify output format  
✅ Test and iterate
