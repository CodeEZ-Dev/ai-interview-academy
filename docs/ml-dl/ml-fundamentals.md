# Machine Learning Fundamentals

## Question

What are the fundamental concepts of Machine Learning?

## Answer

Machine Learning is a subset of AI that enables systems to learn and improve from data without being explicitly programmed.

### ML Types

1. **Supervised Learning** - Learning from labeled data
   - Classification
   - Regression

2. **Unsupervised Learning** - Finding patterns in unlabeled data
   - Clustering
   - Dimensionality Reduction

3. **Reinforcement Learning** - Learning through reward/penalty

### Key Concepts

- **Training Data** - Data used to train model
- **Features** - Input variables
- **Labels** - Target outputs (in supervised)
- **Loss Function** - Measures prediction error
- **Optimization** - Minimizes loss
- **Generalization** - Performance on unseen data

## Key Points

✅ Data quality matters most  
✅ Avoid overfitting  
✅ Validate on test set  
✅ Monitor metrics
